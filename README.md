# 简单点

在手机里管理你的库存和订单。

## 功能介绍

### 仓库管理
- 创建自己的仓库，设置公开或私密
- 公开仓库可被其他用户浏览
- 私密仓库只有创建者可见

### 商品管理
- 在仓库中添加商品（名称、SKU、库存、单位、价格）
- 支持编辑和删除商品
- 库存数量实时更新

### 订单管理
- 买家可以浏览公开仓库
- 将商品加入购物车（支持同商品数量累加）
- 结算购物车中的商品下单
- 支持库存为 0 时也能下单（先下单后生产）

### 订单流程
```
待处理 → 生产中 → 已发货 → 已完成
```
- 卖家：接单生产 → 确认发货
- 买家：确认收货 / 取消订单

### 用户系统
- 用户名密码注册登录
- 退出登录

## 技术栈

- **前端**：微信小程序
- **后端**：微信云开发（云函数 + 云数据库）
- **默认账号**：qc / 1qaz2wsx

## 快速开始

### 1. 导入项目
在微信开发者工具中导入项目目录

### 2. 配置云环境
在 `miniprogram/app.js` 中配置云环境 ID

### 3. 创建数据库集合
在云开发控制台创建以下集合：
- `users` - 用户表
- `warehouses` - 仓库表
- `products` - 商品表
- `orders` - 订单表
- `cart` - 购物车表

### 4. 为集合添加索引
- `users` 集合：为 `username` 字段添加唯一索引
- `warehouses` 集合：为 `owner_id` 字段添加索引
- `orders` 集合：为 `buyer_id`、`warehouse_id` 字段添加索引

### 5. 上传云函数
右键点击 `cloudfunctions` 目录下的每个文件夹，选择「上传并部署」：
- `auth` - 用户认证
- `init` - 初始化（创建默认用户）
- `login` - 登录
- `order` - 订单管理
- `product` - 商品管理
- `warehouse` - 仓库管理

### 6. 运行初始化
在云开发控制台的云函数中运行 `init`，创建默认用户 `qc`

### 7. 开始使用
使用默认账号登录：
- 用户名：qc
- 密码：1qaz2wsx

## 业务逻辑

### 库存管理
- 下单时立即扣减库存
- 取消订单时返还库存
- 库存不会变为负数

### 购物车
- 同商品多次加入，数量自动累加
- 合并显示，页面显示合并后的总数量
- 结算后自动清空已购商品
